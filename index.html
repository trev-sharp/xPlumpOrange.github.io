AdvancedIP | Plugin

This is one of my more recent plugins, and one of my personal favorites. It helps servers track and log players' IP history and geolocations related to the IPs. This plugin is "semi-open source," meaning the basic framework of the plugin is shown, but no actual usable code is given.

Github - https://github.com/xPlumpOrange/AdvancedIP

Spigot - https://www.spigotmc.org/resources/advancedip-1-8-1-12.57118/

Combo Grabber | Java Application

This is an open source Pastebin email:password combo grabber. Uses various HTML skimmers to grab Google search results and raw text data from Pastebin, then automatically deciphers the data and displays it on a custom made GUI. This is a very useful application for account miners.

Github - https://github.com/xPlumpOrange/ComboGrabber

OrangeBot | Java Discord Bot

This is an open source discord bot made in Java. Offers a template for commands, events, etc.. Came with many pre-built commands such as "help," "8ball," "announce," and a few others. This used to be my personal bot for my discord server until I discontinued my discord server.

Github - https://github.com/xPlumpOrange/OrangeBot
