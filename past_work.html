  • AdvancedIP | Plugin

    This is one of my more recent plugins, and one of my personal favorites. It helps servers track and log players' IP history and geolocations related to the IPs. This plugin is "semi-open source," meaning the basic framework of the plugin is shown, but no actual usable code is given.

  • Combo Grabber | Java Application

    This is an open source Pastebin email:password combo grabber. Uses various HTML skimmers to grab Google search results and raw text data from Pastebin, then automatically deciphers the data and displays it on a custom made GUI. This is a very useful application for account miners.

    Github - https://github.com/xPlumpOrange/ComboGrabber

  • Artficial Intelligence API | Java API

    This is my first open source API for Java. This API offers 2 neural netowork objects, a 3 layer and a 5 layer network. It uses the feed foward method and uses back propagation to train. In all honesty, it's a little janky with its hard to understand format and no detailed documentation.

    Github - https://github.com/xPlumpOrange/A.I.-Java-API

  • Trivia Game Helper | Java Application

    This is an open source application that helps answer multiple choice questions. (Originally made for HQTrivia) It uses HTML skimmers to grab Google and Yahoo search results, then uses many layers of checking to find the most likely answer to the question. I even had the audacity to have multi-language support. My all time favorite project.

    Github - https://github.com/xPlumpOrange/Trvia-Game-Helper

  • OrangeBot | Java Discord Bot

    Github - https://github.com/xPlumpOrange/OrangeBot
